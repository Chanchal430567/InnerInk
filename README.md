# 🌸 InnerInk

A warm and secure digital diary application built using the MERN Stack, designed to help users capture memories, thoughts, and emotions in a beautiful and private space.

## Features

-  User Authentication
-  Calendar View
-  Create Diary Entries
-  Edit Entries
-  Delete Entries
-  Search Entries
-  View All Entries
-  MongoDB Database
-  Responsive Design

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- CSS
- JWT Authentication
